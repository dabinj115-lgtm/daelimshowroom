# 대림바스 논현직영쇼룸 안내 페이지

GitHub Pages 업로드용 정적 웹사이트입니다.

## 구성
- index.html
- style.css
- images/logo.png
- images/background.jpg

## GitHub Pages 배포
1. GitHub에서 새 repository 생성
2. 위 파일과 폴더를 그대로 업로드
3. Settings > Pages
4. Source: Deploy from a branch
5. Branch: main / root 선택
6. 발급된 URL 확인

## 버튼 링크 수정
index.html 안의 `<a class="map-button" href="...">` 부분에서 href 값만 바꾸면 됩니다.
